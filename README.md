## Simple fractal in rust


### Запуск 
```bash
cargo run -- <params>
```


### Параметры
```bash
FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
        --dx <dx>         [default: 0.0]
        --dy <dy>         [default: 0.0]
        --zoom <zoom>     [default: 1.0]
```
